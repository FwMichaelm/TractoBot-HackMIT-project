# TractoBot-HackMIT-project
We built a 2-wheel drive robot using SnappyXO pieces to represent a tractor moving through a crop field. We then designed a linear actuator marking mechanism in MotionGen, built it with SnappyXO pieces and mounted it onto the robot. We programmed the robot to navigate a 3x3 grid, stopping at each plant to scan it. We trained an image classification model using Google Teachable Machine and integrated it into the TractoBot iOS app, which runs the model on the iPhone's front camera. When a diseased plant is detected, the app sends a Bluetooth signal to the Arduino, triggering the linear actuator to mark the plant.
